Introduction

Le projet pkg-shadow sur Alioth sert à la fois pour la maintenance de shadow et pour la maintenance du paquet Debian shadow.

Participez !

Si vous voulez participer, vous pouvez nous contacter :

  • Sur notre liste de diffusion (archives),
  • sur IRC (#shadow sur freenode).

Rapport de bogues, patches, demandes de fonctionnalités ou toute forme de retour sont bienvenus sur la liste de diffusion, sur le système de suivi d'Alioth ou le système de suivi des bogues Debian

Les courriers de notiffications de commit sont envoyés sur une liste de diffusion séparée (archives).

Suite de tests

Vous pouvez également contribuer à la suite de tests et améliorer la couverture de ces tests.

La suite de tests est actuellement utilisée dans un environnement Debian. Vous pouvez aider en la testant dans d'autres environnements.
