Willkommen!

Dies ist die Heimatseite der Shadow-Werkzeugsammlung.

2009-07-24 - Neue Veröffentlichung: 4.1.4.2

Eine neue Unterversion wurde veröffentlicht. Lesen Sie shadow-4.1.4.2.NEWS für weitere Details.

2009-05-22 - Neue Veröffentlichung: 4.1.4.1

Dies ist eine neue Unterversion mit Fehlerkorrekturen und einigen kleinen neuen Funktionen. Lesen Sie shadow-4.1.4.1.NEWS für weitere Details.

2009-05-11 - Neue Veröffentlichung: 4.1.4

Diese Veröffentlichung fügt PAM-Unterstützung zu den Werkzeugen Chpasswd und Newusers hinzu. Sie enthält auch Fehlerkorrekturen und kleinere Funktionen, die in shadow-4.1.4.NEWS beschrieben sind.

2009-04-15 - Neue Veröffentlichung: 4.1.3.1

Diese Veröffentlichung behebt einen Fehler, der bei nach der Anmeldung verwandten Terminals zu falsch gesetzten Rechten führte. Nur 4.1.3 war betroffen.

Sie behebt auch einige Fehler in der SE-Linux-Unterstützung.

2009-04-12 - Neue Veröffentlichung: 4.1.3

Eine weitere Version von Shadow wurde veröffentlicht. Sie finden eine umfassende Liste der Änderungen unter shadow-4.1.3.NEWS.

2008-11-23 - Neue Veröffentlichung: 4.1.2.2

Eine neue Version von Shadow wurde veröffentlicht. Sie behebt eine »race condition«-Verwundbarkeit in Login, die zur Erlangung der Eigentümerschaft oder der Modus-Änderung beliebiger Dateien führen konnte sowie eine mögliche Diensteverweigerung (DOS), wenn ein Angreifer gefälschte Einträge in Utmp einschleusen konnte.

Sie finden eine umfassende Liste der Änderungen unter shadow-4.1.2.2.NEWS.

2008-06-26 - Neue Veröffentlichung: 4.1.2.1

Eine neue Version von Shadow wurde veröffentlicht. Sie behebt eine »audit log injection«-Verwundbarkeit in Login, wenn dieses mit Audit-Unterstützung konfiguriert wurde.

Sie finden eine umfassende Liste der Änderungen unter shadow-4.1.2.1.NEWS.

2008-05-25 - Neue Veröffentlichung: 4.1.2

Eine neue Version von Shadow wurde veröffentlicht. Sie können eine umfassende Liste der Änderungen unter shadow-4.1.2.NEWS finden.

2008-04-02 - Neue Veröffentlichung: 4.1.1

Eine neue Version von Shadow wurde veröffentlicht. Sie enthält eine Reihe von Korrekturen für Fehler, die mit der Testsuite gefunden wurden.

2008-03-09 - Die Website ist übersetzt!

Auf der Website wurde eine Internationalisierungsinfrastruktur mittels po4a eingerichtet.

Die Website ist jetzt auf Französisch übersetzt. Sie können über die Links am Ende jeder Seite auf die englische Version zurückspringen.

Die Abdeckung der Testsuite wurde auch zur Website hinzugefügt.

2007-12-10 - Neue Veröffentlichung: 4.1.0

Eine neue Version von Shadow wurde veröffentlicht. Sie enthält Patches aus verschiedenen Distributionen, die Dokumentation der Variablen aus login.defs, Unterstützung für die SHA-Passwortverschlüsselungsmethode und eine Reihe von Fehlerkorrekturen.

Die Handbuchseiten werden jetzt abhängig von der Konfiguration erstellt. Es wird empfohlen, sie bei Änderung von Konfigurationsoptionen neu zu erstellen.

2007-10-28 - Neue Veröffentlichung: 4.0.18.2

Eine neue Version von Shadow wurde veröffentlicht. Sie enthält gesammelte Änderungen aus dem vorherigen Depot und kleinere neuere Änderungen.

Die Paketierung sollte sich nicht groß geändert haben. Teilen Sie uns mit, falls es Paketierungsprobleme gibt.

2007-10-07 - Neues Depot-Layout für die Entwicklung der Originalautoren

Die Entwicklung der Originalautoren der Shadow-Werkzeugsammlung wurde in der Vergangenheit im Rahmen des PLD-Projekts betrieben.

Sie wird jetzt auf Alioth betrieben.

Das Depot von Pkg-Shadow wurde neuorganisiert, um die Pflege des Ursprungsquellcodes von der Pflege des Debian-Pakets zu trennen.

Die Entwicklungsgeschichte von Shadow wurde aus den alten Archiven und dem letzten bekannten Schnappschuss des CVS-Depots auf PLD importiert.
