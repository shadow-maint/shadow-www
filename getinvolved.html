Introduction

The shadow project on GitHub is used for upstream maintenance of shadow.

Get involved!

If you want to participate, you can contact us

  • on our mailing list (archives),
  • on IRC (#shadow on freenode).

Bug reports, patches, feature requests or any feedback are welcomed on the mailing list, in the GitHub Issues or in the Debian BTS.

Commit emails are sent on a separate mailing list (archives).

Testsuite

You can also contribute to the test suite and improve the coverage of these tests.

The testsuite is currently executed on a Debian environment. You can help testing it on another environment.
