Bienvenu !

Vous êtes sur la page d'accueil de la suite d'outils shadow.

2009-07-24 - Nouvelle version : 4.1.4.2

Une version mineure est sortie. Consultez shadow-4.1.4.1.NEWS pour plus de détails.

2009-05-22 - Nouvelle version : 4.1.4.1

Il s'agit d'une version mineure, avec des corrections et quelques petites nouvelles fonctionnalités. Consultez shadow-4.1.4.1.NEWS pour plus de détails.

2009-05-11 - Nouvelle version : 4.1.4

Cette version ajoute la prise en charge de PAM aux outils chpasswd et newusers. Elle contient également des correctif et des fonctionnalités moins importantes qui sont décrites dans shadow-4.1.4.NEWS.

2009-04-12 - Nouvelle version : 4.1.3.1

Cette version corrige un bogue ayant pour conséquence l'application des mauvaises permissions pour les terminaux après une connexion avec login. Il n'affectait que la version 4.1.3.

Elle corrige également des bogues dans la prise en charge de SE Linux.

2009-04-12 - Nouvelle version : 4.1.3

Une autre version de shadow est sortie. Vous trouverez un résumé des modifications sur shadow-4.1.3.NEWS.

2008-11-23 - Nouvelle version : 4.1.2.2

Une nouvelle version de shadow est sortie. Elle corrige une vulnérabilité de type « race condition » dans login, et qui pourrait permettre de devenir propriétaire ou de changer le mode de fichiers ; elle corrige également un une possibilité de dénis de service pour login quand un attaquant peut injecter des entrées dans utmp.

Vous trouverez un résumé des modifications sur shadow-4.1.2.2.NEWS.

2008-06-26 - Nouvelle version : 4.1.2.1

Une nouvelle version de shadow est sortie. Elle corrige une vulnérabilité de type « injection de log d'audit » dans login, quand il est configuré avec la prise en charge des messages d'audit.

Vous trouverez un résumé des modifications sur shadow-4.1.2.1.NEWS.

2008-05-25 - Nouvelle version : 4.1.2

Une nouvelle version de shadow est sortie. Vous trouverez un résumé des modifications sur shadow-4.1.2.NEWS.

2008-04-02 - Nouvelle version : 4.1.1

Une nouvelle version de shadow est sortie. Elle contient de nombreuses corrections de bogues trouvés à l'aide de la suite de tests.

2008-03-09 - Le site web est traduit !

Une infrastructure pour l'internationalisation du site we a été mise en place en utilisant po4a.

Le site web est maintenant traduit en français et vous pouvez revenir à la version anglaise en utilisant les liens en bas de toute les pages.

La couverture des tests de la suite de tests a également été ajoutée au site web.

2007-12-10 - Nouvelle version : 4.1.0

Une nouvelle version de shadow est sortie. Elle contient des patches provenant de différentes distributions, la documentation des variables de login.defs, la gestion des méthodes de chiffrement SHA pour les mots de passe et de nombreuses corrections de bogues.

Les pages de manuel sont désormais générées en fonction de la configuration. Il est recommendé de les re-générer si vous changer les options de configuration.

2007-10-28 - Nouvelle version : 4.0.18.2

Une nouvelle version de shadow est sortie. Elle contient des changements accumulés dans le repository précédent et de petites modification récentes.

Le packaging n'a pas du trop changer. Indiquez-nous si vous trouvez des problèmes.

2007-10-07 - Nouvelle organisation du repository pour le développement amont

Le développement amont de la suite d'outils shadow était géré par le projet PLD.

Il se trouve maintenant sur Alioth.

Le repository pkg-shadow a été réorganisé pour séparer la maintenance amont de la maintenance du paquet Debian.

L'historique de shadow a été importé en utilisant les anciennes archives et la dernière version connue du dépot CVS de PLD.
